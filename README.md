# Ask-Ai-Bar

Vercel-ready static hosting for the Marvin Search prototype.

## Project Structure
- `index.html`: main entry served at `/`
- `marvin-search-prototype.html`: same prototype file, also available at `/marvin-search-prototype`
- `assets/icons/`: icon assets used by the prototype
- `vercel.json`: Vercel route rewrite config

## Deploy on Vercel
1. Push this repo to GitHub.
2. Import the repo into Vercel.
3. Use defaults:
- Framework Preset: `Other`
- Root Directory: `.`
4. Deploy.
